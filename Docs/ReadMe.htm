Description

Version Information Spy displays the version information stored in an executable file (.exe, .dll etc.), or in 32 bit binary resource files.

Several ways of displaying version information are provided:

  • A Windows program that displays version information in its main window, can generate descriptive plain text, XML or HTML reports and can decompile the binary data into the equivalent RC source code.
  • A command line application that can display a textual description of the version information or can decompile to RC source code.
  • A Windows Explorer context menu handler that displays the version information of a selected file in the main application.
  • An Explorer property sheet extension that displays detailed version information in a tab of a file's property sheet.

Compatibility

The program requires Windows 2000 or later. It also requires Internet Explorer v5.5 (and preferably v6) or later. The program does not run on the old Windows 9x platform.

Installation & removal

Administrator Privileges
When installing or uninstalling you need to ensure that you have administrator privileges otherwise the installation can fail.
On Windows Vista and later with UAC prompts enabled, the system will attempt to elevate the process. Standard users will need to provide an administrator password.
Other users will need to run the setup and uninstall programs from an administrator user acccount or "as administrator".

Installation

If you have any early version of Version Information Spy already installed on your system (i.e. any version below version 7.0) you must uninstall the old version before installing the new program.

The installation program for Version Information Spy is named VIS-Setup-x.x.x, where x.x.x is the program's version number. The install program may be distributed in a zip file. If so extract the setup program from the zip file. Run VIS-Setup-x.x.x to start the installation then follow the on-screen instructions.

The program makes the following changes to your system:

  • Files are installed as follows:
    • The main program and help files are copied into an installation folder (\Program Files\DelphiDabbler\VIS7 by default).
    • Documentation is copied to the main installation's Docs sub-folder.
    • Uninstall files are copied to the main installation's Uninst sub-folder.
  • The registry is updated to store information about the program as follows:
    • A COM Server, FileVerCM.dll is registered.
    • The Explorer extensions (context menu handler and property sheet extension) are registered for several file types.
    • The program's help files are registered with Windows.
    • Both the GUI and command line application are registered with Windows.
    • The HKEY_CURRENT_USER\Software\DelphiDabbler\FileVer\7 registry key is created and used to store the program's customisation information.
    • The program's uninstall information is registered with the Add / Remove Programs or Programs and Features control panel applet.
  • A program group is optionally created in the Start Menu.
  • A desktop icon may be created.

Removal

You must close down all Explorer Windows and all Version Information Spy instances before attempting to uninstall the program.

The program can be uninstalled via Add/Remove Programs or Programs and Features accessed from the Control Panel or by choosing the Uninstall Version Information Spy option from the program's start menu group.

Bugs

Please do report any bugs you find. To do this proceed as follows:

  1. Go to the Version Information Tools Issue Tracker on Google Code.
  2. Review the list of reported bugs. If the bug you have found is listed and not fixed please add any information you can to the existing report. If the bug is fixed, update to the latest release.
  3. If the bug has not been reported please add it. Because the bug tracker is used for more that one application be sure to specify that the bug was found in Version Information Spy and set the Project-VIS. label. Please provide as much information as you can including the program version and which version of Windows and Internet Explorer you are using.
