Descriptive Text Report Dialog Box

This dialog box displays a descriptive, plain text, description of the current file's version information. The report is the same as the equivalent report displayed by the command line program.

Clicking the View Externally button displays the report in the default text editor. If no text editor is registered then an error message is displayed.

To save a copy of the report to disk click the Save button. The Copy button stores the report text on the clipboard.

Click Close to dismiss the dialog box.
