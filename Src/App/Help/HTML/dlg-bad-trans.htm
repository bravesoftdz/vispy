Translation Inconsistency Explanation Dialog Box

Providing the facility is enabled (see below), Version Information Spy shows this dialog box when the user has clicked the Explain Problem button on the main user interface. The button appears when an inconsistency in a translation or string table has been detected.

The dialog box displays report on the problem that caused the translation inconsistency. An explanation of the problem is offered along with some resource (RC) source code that shows the changes to the version information necessary to fix the problem.

Click the Close button to dismiss the dialog box.

Enabling the dialog box

If no Explain Problem button is displayed when a translation inconsistency is reported you can switch the feature on as follows:

  • Select the Options | Display Options menu item to show the Display Options dialog box.
  • Choose the Translations tab.
  • Tick the Add explanation button check box.
  • Click OK to accept the changes and close the dialog box.
