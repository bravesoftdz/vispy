Toolbar

The toolbar's buttons provide quick access to a small number of the program's commands. The available buttons are:

Toolbar
button 		Action
Displays a file open dialog box where the desired file can be opened. Files can be opened in other ways. See Opening a File for further information.
Displays the help file's contents page.
Accesses the DelphiDabbler website at the home page. Updates to the program are posted on this site.
Displays the program's about box which shows version and copyright information.

The toolbar can be toggled on and off using the Options | Toolbar menu item.
