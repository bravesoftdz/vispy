Resource Source Code Dialog Box

This dialog displays decompiled resource (RC) source code for the current file's version information. It is displayed when either the Report | Source Code or Report | Corrected Source Code menu options are selected. (The latter is only enabled if the program detects problems with the version information).

The code that is displayed varies slightly depending on which menu option was used to display the dialog box. Here are the differences:

  • Report | Source Code
    The source code is a an estimation of the code that was used to create the resource. If any errors or inconsistencies have been detected the they are noted in comments in the source code but are not fixed.
  • Report | Corrected Source Code
    The errors detected in the source code are corrected and comments are used to identify the changes.

Clicking the View Externally button displays the source code in NotePad.

To save the source code to disk click the Save button. The Copy button stores the code on the clipboard in text format.

Click Close to dismiss the dialog box.
