User Interface

Version Information Spy's main window is separated into three main areas:

  1. The menu bar.
    Use the menu commands to control the program.
  2. The toolbar.
    Buttons that replicate some of the menu commands are displayed on the toolbar. The toolbar is optional and may be hidden. The Options | Toolbar menu option is used to display and hide the toolbar.
  3. The display area.
    This is where details of the current file's version information is displayed. See Displaying Version Information for further details. The display area can be customised using the Options | Display Options menu command to access the Display Options dialog box.

The program's window can be resized in the usual fashion – and its size and placement are remembered between executions.

Version Information Spy can optionally be activated from Windows Explorer by selecting "Version Information" from the menu that appears when an executable file is right clicked. The program also adds a new page to Explorer's property dialog box for files that contain version information. See the Using the Explorer Extensions topic for more information.
