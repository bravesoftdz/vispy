Display Options Dialog Box

This tabbed dialog box is used to customise Version Information Spy's appearance. It is accessed from the Options | Display Options menu item. The four tabs are:

Pop-ups

Used to determine whether a pop-up hint is displayed when the cursor hovers over information that extends beyond the edge of the display. If the Display overflow text in pop-up window check box is checked these pop-up windows are displayed (the default). Clear the check box to prevent pop-ups.

Fixed File Info

This tab is used to configure how fixed file information is displayed. There are three options, each selected via a check box:

  • Display structure information
    Two lines of information that describe the fixed file information record structure can be displayed   the structure's version number and a "signature". This information is the same for every program, so the information is of little value and is hidden by default. Tick the check box to display the information.
  • Display creation date
    Programmers may store a program's creation date in a fixed file information field, although this is rarely done. Version Information Spy can display this field, which is hidden by default. Tick the check box to display the creation date.
  • Describe file flags
    The File Flags and File Flags Mask fixed file information fields are bitmasks that can either be displayed as hex values or as natural language descriptions of the flags. Check the box to display the natural language description and clear it to display the value in hexadecimal.

Translation

Several programs contain version information that departs from the Microsoft specification which states that for each translation there should be a matching string table. Version Information Spy can detect any deviation from the standard and can highlight or provide additional information about such inconsistencies. Use the following controls to customise how these errors are reported:

  • Highlight inconsistent translation entries
    When this check box is ticked inconsistent translations are highlighted in the translation drop-down list using the colour specified by the Highlight colour colour selector.
  • Explain translation inconsistencies
    A brief explanation of any translation inconsistencies is displayed above the translation drop-down list if this check box is ticked.
  • Add explanation button
    If this box is checked a button is displayed above the translation drop-down list whenever an inconsistent translation is selected. Clicking the button displays detailed information about the problem along with a suggestion as to how to fix it.

String Info

This tab is used to configure how the string file information is displayed. While most software names the strings in this section according to the standard recommended by Microsoft it is possible (and quite common) to include arbitrary string names in string tables.

Version Information Spy can highlight such names in the colour specified in the Highlight colour colour selector. You must check the Highlight non-standard string information items check box to enable this feature.

Other Controls

You can restore all the display options to their default values by clicking the Restore Defaults button.

All changes can be aborted by clicking the Cancel button. Click OK to accept changes. Changes are immediately reflected in the display.

The dialog box remembers the tab that was in use when the dialog is closed and displays the same tab again when the dialog is next opened.
