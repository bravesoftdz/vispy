The Version Extra Property Sheet

This property sheet is one of the extensions added to Explorer by Version Information Spy. The property sheet is added to the standard Explorer Properties dialog box that can be displayed from Explorer's pop-up menu. The property sheet is only displayed for executable files of types registered with Version Information Spy.

The property sheet provides more detailed version information than is supplied in the Properties dialog box by default. The information displayed has a similar layout to the Version Information Spy program's main display, but does not support all the highlighting and reporting features of the main program.

The information displayed is:

  • Fixed file information at the top of the property sheet.
  • Details of all the translations (languages and character sets) used by the program. This information appears in the drop down list in the middle of the property sheet. There is usually only one translation, but there may be more.
  • A table of String information at the bottom of the property sheet. There should be one string table for each translation. Selecting a translation will display any associated string table.

If the full facilities of the main program are required then clicking the Advanced button starts the main program and displays the file's version information in it.
