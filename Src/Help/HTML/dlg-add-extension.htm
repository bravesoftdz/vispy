Add Extension Dialog Box

This dialog box is displayed when the Add button of the Explorer Extensions dialog box is clicked. It is used to select or enter an extension that will be added to the list of extensions supported by the program's Explorer extensions.

Either select an extension from the list of extensions known to Windows or enter one in the edit box then press OK to add the extension or Cancel to abort the process.

Note that extensions already known to the explorer extensions are displayed in grey and the OK button is disabled if one of these extensions is selected.
