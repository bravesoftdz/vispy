Overview

Version Information Spy allows you to see details of the version information stored in an executable or 32 bit binary resource file. It also allows you to check the integrety of the data and to create reports and decompile to RC source code.

  • Read the program's feature list

This help file focusses on helping you to use Version Information Spy while also giving some background on the data stored inside a program's version information.

Key tasks

  • Learning the user interface
  • Opening a file
  • Creating reports
  • Recreating RC source code
  • Dealing with errors and inconsistencies
  • Using the Explorer extensions
  • Using the command line program
