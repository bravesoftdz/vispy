XML Report Dialog Box

This dialog box displays the content of the current current file's version information in XML format. Any inconsistencies in the file are not noted – this is purely a description of the version information as read.

To save a copy of the report in XML format to disk click the Save button. The Copy button stores the XML on the clipboard in text format.

Clicking the View button displays the XML report in Notepad.

Click Close to dismiss the dialog box.
