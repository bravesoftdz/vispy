Using the Explorer Extensions

When Version Information Spy is installed a Windows Explorer shell extension is provided. The extension provides two facilities when a file containing version information is selected in Explorer.

Explorer Context Menu

A new Version Information menu option is added to the Explorer context menu. Selecting this menu option opens Version Information Spy and displays the file's version information.

The context menu item only appears for files with extensions that are known to Version Information Spy. Further extensions can be added and activated using the Advanced tab of the Explorer Extensions dialog box. Alternatively the program can ask the user whether to activate the context menu for a previously unknown file type when such a file is opened.

By default a single instance of the program is used for all files accessed via this context menu, but this can be changed.

Property Sheet Enhancement

Version Information Spy can also add a new Version Extra tab to a file's property sheet. This new tab provides detailed version information and has a similar display layout to the main program.

Like the context menu handler, the property sheet is only available for files with known extensions. The Advanced tab of the Explorer Extensions dialog box can also be used to add further file extensions. The automatic registration of unknown file extensions noted above also works for the property sheet.

Customising the Shell Extensions

The behaviour of both the context menu and the property sheet can be customised, or switched on or off, by using the Explorer Extensions dialog box. The dialog box is displayed by selecting the Options | Explorer Extension menu option.
